# Dashboard

The Next.js 16 web application that powers the Twitter Archive Explorer. Built with React 19 and Tailwind CSS 4.

## Quick start

```bash
npm install   # first time only
npm run dev   # starts at http://localhost:3000
```

A preflight check runs automatically before the dev server starts, validating dependencies and build cache.
